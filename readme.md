# Social Media
## Installation

Clone the project

```bash
  git clone https://github.com/maheswar-dileep/KDC-social-media-task.git
```
Install dependencies

```bash
  npm install
```

Add environment variables

```bash
  touch .env
``` 
add environment variables

```bash
MONGODB_URL = <--mongodb_url-->
PORT = <---port--->

example 

MONGODB_URL = "mongodb://127.0.0.1:27017/social-media"
```
## Run Locally


Start the server

```bash
  npm start
```
